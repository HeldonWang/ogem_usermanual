Parameter

Syntax

%m_u_binom_odds (
inds=<input-dataset>,
whr=<where-condition>,
pop_flag=<population-flag-condition>,
trtgrpn=<numeric-treatment-group-variable>,
trtgrp=<treatment-group-variable>,
trtpair=<comparison-group-pair>,
trtfmt=<treatment-display-format>,
trtref=<reference-treatment-group>,
strata=<statification-variable>,
var=<analysis-variable>,
event=<positive-event-value>,
alpha=<alpha-level>,
pvalside=<pvalue-sides>,
pvdecim=<pvalue-decimal-place>,
cidecim=<confidence-interval-decimal-place>,
relcal=<relaive-risk-display-Y/N>,
cltype=<CI-calculation-type>,
supds=<dummy-dataset-Y/N>,
missval=<missing-value-display-text>,
ord=<value-of-order>,
debug=<Y|N>
);

Control input

specifies the input data set.

Item Contents
Default
Value A one or two level dataset name.
Requirement Mandatory
Restriction
Interaction

work.anl

Item Contents
Default
Value
Requirement Optional
Restriction
Interaction
Eg. paramcd='TRORESPU'

specifies the population flag and condition that will be applied onto adsl and inds analysis dataset.

Item Contents
Default
Value
Requirement Mandatory
Restriction
Interaction
Eg. ittfl='Y'

Control analysis

trtgrpn

specifies the variable name of numeric value of treatment group.

Item Contents
Default
Value
Requirement Mandatory
Restriction The variable type of trtgrpn must be numeric.
Interaction
Eg. trt01pn

trtgrp

specifies the variable name of treatment group.

Item Contents
Default
Value
Requirement Mandatory
Restriction The variable type of trtgrp must be chracter.
Interaction
Eg. trt01p

specifies the comparison treatment groups that needs to be analyzed.

Item Contents
Default
Value
Requirement Required
Restriction The treatment groups specified must be within the analysis dataset.
Interaction
Eg. str(1,2)
1 2

trtfmt

specifies the format applied to the numeric treatment group if trtgrpn != trt01pn.

Item Contents
Default
Value Predefined format name
Requirement Optional
Restriction
Interaction
Eg. example_format

specifies the reference treatment group.

Item Contents
Default
Value
Requirement Required
Restriction Variable type for trtref must be numeric.
Interaction
Eg. 3

strata

specifies the stratification variables if stratification is required for analysis.

Item Contents
Default
Value
Requirement Optional
Restriction Variables specified by strata must have type of numeric.
Interaction
Eg. strata1n strata2n

var

specifies the variable that will be analyzed on.

Item Contents
Default aval
Value
Requirement Optional
Restriction Variable specified by var must have value of 1/0.
Interaction
Eg. aval

Default 1

Requirement Optional
